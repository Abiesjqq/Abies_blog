## 位置与速度


> 当一个坐标系 ({B}) 相对另一个坐标系 ({A}) 运动时，某个点 (Q) 的速度应该怎么表示？

在机器人学里，末端执行器、连杆、关节坐标系都在运动，所以只会写位置是不够的，还要知道这些位置随时间怎么变化。

---

## 1. 位置向量的微分：速度是什么？

假设点 (Q) 在坐标系 ({B}) 中的位置为

[
{}^B Q
]

那么它关于时间的导数就是点 (Q) 在坐标系 ({B}) 中观察到的速度：

[
{}^B V_Q = \frac{d}{dt}{}^B Q
]

也就是：

[
{}^B V_Q
========

\lim\_{\Delta t \to 0}
\frac{{}^B Q(t+\Delta t)-{}^B Q(t)}{\Delta t}
]

这里的关键点是：**速度不只是“一个点位置的导数”，还要说明在哪个坐标系中求导、在哪个坐标系中描述。**

比如：

[
{}^A V_Q
]

表示点 (Q) 的速度用坐标系 ({A}) 的坐标轴来描述。

而

[
{}^B V_Q
]

表示同一个点的速度用坐标系 ({B}) 的坐标轴来描述。

它们可以表示同一个物理速度，但坐标数值不同。

---

## 2. 点的位置在两个坐标系之间如何转换？

如果点 (Q) 在 ({B}) 中的位置是

[
{}^B Q
]

坐标系 ({B}) 相对 ({A}) 的原点位置为

[
{}^A O_B
]

旋转矩阵为

[
{}^A_B R
]

那么点 (Q) 在 ({A}) 中的位置是：

[
{}^A Q
======

{}^A O_B

- {}^A_B R,{}^B Q
  ]

含义如下：

[
{}^A Q
]

等于：

1. 先走到 ({B}) 的原点；
2. 再从 ({B}) 原点走到点 (Q)，但这个向量要先从 ({B}) 坐标转换到 ({A}) 坐标。

这就是正运动学里反复用到的基本关系。

---

## 3. 如果坐标系只有平移，没有旋转

先考虑简单情况：({B}) 相对 ({A}) 只平移，不旋转。

这时旋转矩阵

[
{}^A_B R
]

不随时间变化。

对

[
{}^A Q
======

{}^A O_B

- {}^A_B R,{}^B Q
  ]

两边求导，有：

[
{}^A V_Q
========

{}^A V\_{BORG}

- {}^A_B R,{}^B V_Q
  ]

其中：

[
{}^A V_{BORG}
]

是坐标系 ({B}) 原点相对于 ({A}) 的速度。

这个公式很直观：

[
Q\text{ 在 }A\text{ 中的速度}
========================

B\text{ 原点的速度}

- Q\text{ 相对 }B\text{ 的速度转换到 }A
  ]

如果 (Q) 固定在刚体上，那么

[
{}^B V_Q = 0
]

于是：

[
{}^A V_Q
========

{}^A V\_{BORG}
]

也就是说，如果刚体只是整体平移，刚体上所有点的速度都一样。

---

## 4. 角速度向量是什么？

接下来考虑旋转。

刚体的旋转可以用角速度向量表示。假设坐标系 ({B}) 相对 ({A}) 转动，则角速度写作：

[
{}^A \Omega_B
]

它有两个含义：

1. **方向**：表示瞬时转轴的方向；
2. **大小**：表示绕这个瞬时转轴旋转的快慢。

例如：

[
{}^A \Omega_B
=============

\begin{bmatrix}
\Omega_x\
\Omega_y\
\Omega_z
\end{bmatrix}
]

表示 ({B}) 相对 ({A}) 的角速度，用 ({A}) 的坐标轴来描述。

注意：

[
{}^A \Omega_B
]

不是普通的欧拉角导数。它是一个真实的物理角速度向量。

---

## 5. 旋转矩阵的导数为什么会引出反对称矩阵？

旋转矩阵是正交矩阵，满足：

[
R^T R = I
]

对时间求导：

[
\dot{R}^T R + R^T \dot{R} = 0
]

令

[
S = \dot{R}R^T
]

或者在另一种形式下：

[
S = R^T \dot{R}
]

可以得到：

[
S^T = -S
]

也就是说，(S) 是一个反对称矩阵。

三维空间中，任何反对称矩阵都可以和一个三维向量一一对应。

对于角速度向量

[
\Omega =
\begin{bmatrix}
\Omega_x\
\Omega_y\
\Omega_z
\end{bmatrix}
]

对应的反对称矩阵是：

[
\Omega^\wedge
=============

\begin{bmatrix}
0 & -\Omega_z & \Omega_y\
\Omega_z & 0 & -\Omega_x\
-\Omega_y & \Omega_x & 0
\end{bmatrix}
]

这个矩阵的作用是把叉乘写成矩阵乘法：

[
\Omega \times P
===============

\Omega^\wedge P
]

也就是说：

[
\Omega^\wedge P
]

和

[
\Omega \times P
]

完全等价。

---

## 6. 刚体一般运动时，点的速度公式

如果 ({B}) 相对 ({A}) 既平移又旋转，那么点 (Q) 的速度是：

[
{}^A V_Q
========

{}^A V\_{BORG}

- {}^A \Omega_B
  \times
  \left({}^A_B R,{}^B Q\right)
- {}^A_B R,{}^B V_Q
  ]

这个公式非常重要。

逐项解释：

[
{}^A V_Q
]

是点 (Q) 在 ({A}) 中的速度。

第一项：

[
{}^A V_{BORG}
]

表示坐标系 ({B}) 原点的平移速度。

第二项：

[
{}^A \Omega_B
\times
\left({}^A_B R,{}^B Q\right)
]

表示由于坐标系 ({B}) 旋转导致点 (Q) 产生的速度。

第三项：

[
{}^A_B R,{}^B V_Q
]

表示点 (Q) 自己在 ({B}) 中运动产生的速度，再转换到 ({A}) 中。

---

## 7. 如果点固定在刚体上，公式会简化

如果点 (Q) 固定在刚体 ({B}) 上，那么它在 ({B}) 中的位置不变：

[
{}^B V_Q = 0
]

于是速度公式变成：

[
{}^A V_Q
========

{}^A V\_{BORG}

- {}^A \Omega_B
  \times
  \left({}^A_B R,{}^B Q\right)
  ]

这就是刚体运动里最常见的速度公式：

[
\text{点速度}
==========

\text{原点平移速度}

- \text{旋转引起的速度}
  ]

也可以理解为：

[
v_Q = v_O + \omega \times r
]

其中 (r) 是从刚体参考点 (O) 指向点 (Q) 的位置向量。

---

## 8. 角速度在不同坐标系中的转换

角速度是向量，所以也可以换坐标系描述。

如果角速度在 ({B}) 中表示为

[
{}^B \Omega_C
]

那么在 ({A}) 中表示为：

[
{}^A \Omega_C
=============

{}^A_B R,{}^B \Omega_C
]

更一般地，如果有三个运动坐标系 ({A})、({B})、({C})，则角速度满足叠加关系：

[
{}^A \Omega_C
=============

{}^A \Omega_B

- {}^A_B R,{}^B \Omega_C
  ]

含义是：

[
C\text{ 相对 }A\text{ 的角速度}
=========================

B\text{ 相对 }A\text{ 的角速度}

- C\text{ 相对 }B\text{ 的角速度转换到 }A
  ]

这个公式后面推导机器人连杆角速度递推公式时会用到。

---

## 9. 本部分最重要的公式总结

### 位置变换

[
{}^A Q
======

{}^A O_B

- {}^A_B R,{}^B Q
  ]

### 一般速度公式

[
{}^A V_Q
========

{}^A V\_{BORG}

- {}^A \Omega_B
  \times
  \left({}^A_B R,{}^B Q\right)
- {}^A_B R,{}^B V_Q
  ]

### 固定在刚体上的点

[
{}^A V_Q
========

{}^A V\_{BORG}

- {}^A \Omega_B
  \times
  \left({}^A_B R,{}^B Q\right)
  ]

### 角速度坐标变换

[
{}^A \Omega_C
=============

{}^A_B R,{}^B \Omega_C
]

### 角速度叠加

[
{}^A \Omega_C
=============

{}^A \Omega_B

- {}^A_B R,{}^B \Omega_C
  ]

### 叉乘矩阵

[
\Omega^\wedge
=============

\begin{bmatrix}
0 & -\Omega_z & \Omega_y\
\Omega_z & 0 & -\Omega_x\
-\Omega_y & \Omega_x & 0
\end{bmatrix}
]

[
\Omega \times P = \Omega^\wedge P
]

---

这一部分的核心思想是：

> 刚体上某点的速度由三部分组成：坐标系原点的平移速度、刚体旋转带来的速度、点相对该坐标系自身运动产生的速度。
