## MIMD

ILP -> TLP -> MIMD

内存访问方式：

- Shared memory：共享统一的地址空间
- Message passing：通过ICN连接

NORMA

### UMA

一致性访问：memory对于PE而言均匀分布

每个PE可另加cache或private memory。

### UNMA

非一致性：访问时间不一致，访问local的memory更快。

改进：NC-NUMA和CC-NUMA，后者需要cache一致性协议。

COMA

### Cache一致性问题

- Cache coherence：一个cache写后，其他cache读不一致
- Memory consistency：写入的值什么时候能被正确地读

怎么解决cache coherence问题？

- UMA：snoopy coherence protocols
- NUMA：directory protocol
