## SIMD

SIMD（Single Instruction Multiple Data）指一条指令同时处理很多数据，适合矩阵运算。常见有向量处理器和阵列处理器。

### 向量处理器

向量运算中每个元素之间通常互不相关，因此适合流水线。当循环之间没有前后依赖关系时，可以展开成向量。

三种向量处理方式：

1. **Horizontal Processing**（水平处理）：逐个元素计算。但同一运算中，可能有RAW依赖和功能切换，效率低。
2. **Vertical Processing**（垂直处理）：整体计算中间向量。按中间向量存储位置，进一步分为：
   - Memory-Memory向量机：存在内存。不用。
   - Register-Register向量机：所有向量都存在向量寄存器中。需要考虑硬件中寄存器数量。
3. **group Processing**：综合水平和垂直处理。将原向量分成小的向量组，先组内纵向计算，再组间横向计算。

CRAY-1是计算机历史上的经典向量机。

**_Q：怎么提升向量处理器的性能？_**

可以使用多个功能部件并行、多处理器执行。

**Vector Chaining**（向量链接）是流水线思想的进一步扩展。前一条指令刚算出一个元素（未全部执行完），立即送入后续指令执行。当两指令间存在数据依赖（RAW）、但不存在功能部件冲突时，可用chaining。

当向量长度超过寄存器容量时，可使用**segmented vector**（分段向量），即分成固定长度的小向量。通常由硬件自动完成。

**_Q：怎么计算向量处理器操作时间？_**

1. 设向量长度为 $N$，单条指令执行时间如下，其中第一个括号为第一个元素产生的时间，$N-1$ 为后续元素依次输出的时间：

$$
(1 + \text{功能部件时间} + 1) + (N-1)
$$

2. 如果有操作并行，则将并行部分视为整体，取各指令时长的最大值。
3. 如果有vector chaining，则后续每个元素所占时间为1。

### 阵列处理器

Array Processor（阵列处理器）指，大量处理单元（PE, processing element）组成阵列，在同一个控制单元控制下同时执行同一条指令。

两种存储结构：

1. **Distributed Memory**（分布式内存）：每个PE都有自己的内存。快，但通信复杂。
2. **Centralized Shared Memory**（集中共享内存）：所有PE共享内存。通信方便，但容易冲突。

Parallel computer 的核心是通信结构，即PE间的通信。互连网络（ICN，Interconnection Network）用于PE间数据交换。

### 互连网络

ICN的组成为：CPU、memory、interface（网络接口）、link（物理链路）和switch node（交换节点，决定数据传输方向）。

网络分为两种：

1. **Static Network**（静态网络）：节点间连接固定。
2. **Dynamic Network**（动态网络）：节点间连接可变化。

#### 单级互连网络

单级互连网络（Single-stage Interconnection Network）指只有一层连接关系的网络，从入端通过互连函数直接映射到出端。

设出入端个数为 $N$，表示为二进制需要 $n$ 位。常见的节点互连函数为：

- **Cube network**：共 $n$ 个cube function，其中第i个为将入端二进制的第i位取反。互连方式形成立方体或超立方体。任意两节点能通过 $n$ 步到达，因为最多改 $n$ 个位。
- **PM2I network**（Plus/Minus 2^i）：共 $2n$ 个PM2I function（其中两个等价），其中第i个作用下，节点 $j$ 能对应 $j\pm 2^i\mod N$。
- **Shuffle exchange network**：共 $n$ 个exchange函数和 $n-1$ 个shuffle函数，其中shuffle为二进制左移一位，exchange为翻转最低位。任意两节点，能通过 $2n-1$ 步到达。

静态网络的基础网络拓扑结构有：Loop with chord array, extensions of tree array, star array, grid, 2D torus, hypercube, cube with loop。

动态网络连接可变，可通过bus、crossbar（交叉开关）实现。多级互连网络也是一种实现方式。

#### 多级互连网络

网络规模大时，将多个单级互连网络级联，形成多级互连网络（Multi-stage Network）。它通过switch unit（交换单元）实现，switch unit有2x2通路，分为straight（上上，下下）和exchange（上下，下上）两种状态。

- **Cube network**
- **Omega network**：多层shuffle和switch。

## SIMT

GPU为大量SIMD内核+多线程调度。CUDA是NVIDIA的GPU编程模型，其核心思想为统一所有并行为CUDA thread。Data element组成thread，thread组成block，block组成grid。按照GPU memory hierarchy存储。

**SM**（Streaming Multiprocessor）是GPU的基本结构，每个SM负责执行多个thread blocks。
