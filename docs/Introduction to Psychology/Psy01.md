# SPOC总结

## 人格与心理

人格的特征：

1. 完整性：人格作为整体而言
2. 稳定性：人格对时间、空间稳定（这样才能人格测量），但在一定程度上可发展
3. 独特性：不存在完全相同的人格，也是沟通障碍的开源
4. 社会性：人格依赖于与他人的交往
5. 动力性：会因为人格而做出某些行为

Q：人格和性格的区别？

A：性格表示社会评价部分，有好坏之分，通过后天形成

人格特质理论：人格分为不同特质，各种特质组成完整人格

人格测试：根据人格特质理论设计问题，得到各种特质的表现。常见有卡特尔16相人格测试（16PF）。其他人格理论（如精神分析，人本主义，行为主义）对应不同的测试，或不支持测试。

主观验证效应：接收信息 --> 寻找事例验证 --> 确认信息

人格倾向是正常的，每个人至少有一种人格倾向；人格倾向+痛苦程度和统计极端性，可能是人格障碍。

## 动机与行为

马斯洛需要层次理论：需要是分层次的，只有当底层需要得到满足或部分满足时，才会追求高层的需要。生理 --> 安全 --> 归属和爱 --> 尊重 --> 自我实现。低两层为缺失需要，高三层为发展需要。

人本主义：人能充分发挥潜能，成为他想成为的任何人。马斯洛：凡自我实现者，必有高峰体验。

人类学习的方式：经典条件反射（巴甫洛夫的狗，先刺激再反应），操作条件反射（斯金纳的小鼠，先反应再刺激），观察学习（模仿）

奖励和惩罚有不同的形式。e.g.鼓励vs表扬。

传统行为主义：人格和行为由环境塑造，人格是人内外行为的综合，个体差异是因为外界强化路径不同。否定了人的能动性。

社会学习理论：个人、行为、环境三者交互。替代学习/观察学习（没有实际行动，通过观察推理学习）

自我效能感：是否相信自己能做到。有言语劝说、实际体验、替代性经验（别人家孩子，只有当替代性经验和自身有相似性时，才能带来自我效能感）

人的很多行为是由社会情景决定的。e.g.从众行为，有信息性从众（学习做法）和规范性从众（群体压力），从众的发生和群体的约束力和吸引力有关。米尔格伦电击实验。

海德 归因理论：行为的主导性原因区分你在于人、情景、责任。行动者-观察者错误指两者往往有不同的归因。第一类归因为内因、外因，第二类归因为稳定性因素、不稳定因素。

协变理论：两个事件出现与否有同步性，则会将一个视为另一个的原因。

自利性偏差：倾向于对自己有利。自我中心偏差：期望别人按和自己相同的决定行动。

责任分散理论：在寻求他人帮助的情景下，帮助的人越多，每个人的责任越小，受帮助的概率越小。

助人五步走：觉察，解释，责任，能力，行动。受自然环境、社会环境、榜样、匆忙程度、助人技能影响。

态度：认知成分（实用性，知识）、情感成分、行为成分（做出行为后，调整态度）。态度的改变与信息来源、信息因素、接收者有关。

社会助长：个体对他人的意识，影响行为的效率（可能提高，可能降低，和对活动的掌握程度有关）。e.g.主客场。

社会懈怠：团体一起活动时，个体做出 努力比单独活动更少。

去个性化：身份隐匿、责任模糊时，行为可能与原有人格有较大差异。

## 爱情与动机

爱情类似一种成瘾反应，和多巴胺等化学物质共同作用。

互补性是爱情的原因吗？不一定。性格差异更容易被感知到，但真正可能起作用的共同兴趣等不易被发现。

社会认知指对人或其行为的认识。社会认知整合形成社会印象，社会印象有间接性和固执性（经过自己整合）、综合性。

- 首因效应：最先的印象对认知有极其重要的影响（第一印象，一见钟情/不钟情）。连续互动中占比大。
- 近因效应：最近的印象对认知有极其重要的影响。断续互动中占比大。
- 晕轮效应：对某个人形成好或坏的印象后，对其他行为会朝这个方向解释。
- 刻板效应：针对群体。

认知失调时，会通过改变认知、增加一致性认知、将一致性认知视为更重要、减少选择感，来减少认知与现实的偏差。e.g.助人效应，帮助别人后喜欢ta。

## 情绪与情感

情绪和情感的产生：由客观事物激发，在一定的需要下产生，但其性质由人的认知决定。

情感表达中，面部肌肉是很重要的一环（微表情）。

女性的共情能力一般比男性更强。共情和同情不同。共情需要尊重、倾听、生活阅历、接纳。

应激源：压力的来源，分外在和内在、具体和非具体。应激状态下，有报警阶段、阻抗阶段、耗竭阶段。
