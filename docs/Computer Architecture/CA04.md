## Virtual Memory怎么工作？

四个问题。

**_Q：块在内存中怎么对应？_**

使用全相联，减少冲突。

**_Q：怎么在内存中找对应的块？_**

VA为VPN+offset，VPN经过page table得到PPN，PPN+offset组成PA。

**_Q：冲突时怎么替换？_**

LRU。

**_Q：怎么写？_**

write back。

**_Q：怎么更快？_**

一次访问要两次访存，怎么更高效？TLB（translation looksaside buffer）。TLB的entry中，tag为VPN，data为PPN，valid bit，use bit，dirty bit等。

**_Q：Page size对性能的影响？_**

- page size大：page table小，占用内存少；每次transfer时间长，但能调换更多的数据
- page size小：节省storage

实际使用multiple page sizes
